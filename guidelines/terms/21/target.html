target

New

region of the display that will accept a touch action

If a portion of a touch target that does not perform the same action or go to the same page is overlapped by another touch target such that it cannot receive touch actions, then that portion is not considered a touch target for purposes of touch target measurements.

Proposed modification: Region of the display that will accept a pointer action. If a portion of a target that does not perform the same action or go to the same page is overlapped by another target such that it cannot receive pointer actions, then that portion is not considered part of the target for purposes of target measurements.
